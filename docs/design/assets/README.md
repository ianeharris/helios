# Helios Asset Library

This library keeps visual decisions traceable. It is deliberately small at the start: a curated reference index now, then Bradgate-specific material as it becomes available.

## Folders and provenance

- `visual-reference-index.md` is the source list for external product references. It links to the source page rather than copying third-party imagery into the repository.
- Future owned Bradgate photographs, floor-plan material and approved camera stills go under `owned/` with a short source note and capture date.
- Future generated imagery goes under `generated/` with the original prompt, creation date and intended use recorded beside it.
- Licensed third-party assets may be added only when their licence and source URL are recorded in the same folder.

Do not add scraped marketing imagery, vendor logos or screenshots solely because they are easy to download. The library is for design review and production-ready assets, not an untraceable inspiration folder.
