# Visual Reference Index

Last reviewed: 2026-07-10

These are product-study references, not a feature-parity checklist. They are linked rather than copied so provenance remains clear and the collection can be refreshed as the products evolve.

| Reference | Study for | Avoid copying |
|---|---|---|
| [Nice ELAN Control](https://www.niceforyou.com/na/solutions/control-systems) | Calm whole-home composition, room-first information architecture and a consistent control language. | Brand styling, proprietary hardware assumptions and any central-service dependency. |
| [Savant energy experience](https://www.savant.com/power/savant-app/) | Energy as a daily decision surface: solar, battery, grid, history, energy modes and notifications. | Savant-specific electrical-panel features that do not map to Fox ESS or Octopus. |
| [Control4 X4 app](https://www.control4.com/get-control4-appx) | Fast repeat-use controls, favourites, whole-home status, routines and personalised home-screen density. | A generic widget catalogue or an entertainment-first hierarchy. |
| [Crestron Home UI](https://docs.crestron.com/en-us/8525/Content/CP4R/Operation/Operation.htm) | Restrained home-to-room navigation that can scale across mobile, tablet and fixed panels. | The assumption that every view needs the same density or a dedicated touch panel. |

## Bradgate design questions this index supports

- What belongs in the persistent whole-home context on every device?
- What does a room need to show before a control is touched?
- Which actions should be scene-first, and which deserve direct control?
- How does the Energy screen make an export, charging or tariff decision clear at a glance?
- What changes between phone, tablet and workstation composition while preserving the same navigation and terminology?

## Review method

Use the linked sources and the current prototype together. Extract patterns, not screenshots: record the observed pattern, the Helios adaptation and the reason it is appropriate for Bradgate before it becomes a production requirement.
